     เรื่องน่ารู้ของ Python

  • โหมดอิมมีเดียท (immediate mode)

    เป็นโหมดที่ผู้ใช้จะพิมพ์คำสั่งลงไปในส่วนที่เรียกว่าเชลล์ (shell) หรือคอนโซน (console) ทีละคำสั่ง และตัวแปลภาษาจะแปลคำสั่ง หากไม่มีข้อผิดพลาดจะทำงานตามคำสั่งดังกล่าว

  • โหมดสคริปต์ (script mode)

    นโหมดนี้ผู้เขียนโปรแกรมต้องพิมพ์คำสั่งหลายคำสั่งประกอบกันแล้วบันทึกเป็นไฟล์ไว้ก่อน เพื่อจะสั่งให้ตัวแปลภาษาทำงานตามคำสั่งตั้งแต่คำสั่งแรก จนถึงคำสั่งสุดท้าย

  • คำอธิบายโค้ด (Comment)

    มักเขียนแทรกไว้ในโค้ดเพื่อกลับมาดูทีหลังจะได้รู้ว่าโค้ดส่วนนั้นเอาไว้ทำอะไร หรือเอาไว้ให้ผู้ที่รับช่วงพัฒนาโปรแกรมต่อดู เพื่อจะได้รู้ความหมายหรือจุดมุ่งหมายในการเขียนโค้ดส่วนนั้น ๆ

Python History

-ประวัติไพธอน-

     ภาษาไพธอน (Python programming language) เป็นภาษาโปรแกรมแบบอินเทอร์พรีเตอร์ ที่สร้างโดย กีโด ฟาน รอสซัม (Guido van Rossum) ในพ.ศ. 2533 ปัจจุบันดูแลโดย มูลนิธิซอฟต์แวร์ไพธอน

Why Python?

-จุดเด่นของภาษาไพธอน-

1.เป็นภาษาสคริปต์

     เนื่องจากภาษาสคริปต์ ซึ่งทำให้ใช้เวลาในการเขียนและคอมไพล์ไม่มาก จึงเหมาะกับงานด้านการดูแลระบบ (System administration) เป็นอย่างยิ่ง รวมทั้งมีการสนับสนุนภาษาไพทอนโดยเป็นส่วนหนึ่งของระบบปฏิบัติการยูนิกซ์, ลินุกซ์ และสามารถติดตั้งให้ทำงานเป็นภาษาสคริปต์ของวินโดวส์ ผ่านระบบ Windows Script Host ได้อีกด้วย

2.ไวยากรณ์อ่านง่าย

     เนื่องจากไวยากรณ์ของไพธอนได้กำจัดการใช้สัญลักษณ์ที่ใช้ในการแบ่งบล็อกของโปรแกรม และใช้การย่อหน้าแทน ทำให้สามารถอ่านโปรแกรมที่เขียนได้ง่าย นอกจากนั้นยังมีการสนับสนุนการเขียน docstring ซึ่งเป็นข้อความสั้นๆ ที่ใช้อธิบายการทำงานของฟังก์ชัน, คลาส, และโมดูลอีกด้วย

3.ความเป็นภาษากาว (Glue Language)

     เนื่องจากสามารถเรียกใช้ภาษาโปรแกรมอื่นๆ ได้หลายภาษา ทำให้เหมาะที่จะใช้เขียนเพื่อประสานงานโปรแกรมที่เขียนในภาษาต่างกันได้

4.ไลบรารีในไพธอน

   การเขียนโปรแกรมในภาษาไพทอนโดยใช้ไลบรารีต่าง ๆ เป็นการลดภาระของโปรแกรมเมอร์ได้เป็นอย่างดี ทำให้โปรแกรมเมอร์ไม่ต้องเสียเวลากับการเขียนคำสั่งที่ซ้ำๆ เช่นการแสดงผลข้อมูลออกสู่หน้าจอ หรือการรับค่าต่าง ๆ ไพทอนมีชุดไลบรารีมาตรฐานมาให้ตั้งแต่ติดตั้งอินเตอร์พรีเตอร์ นอกจากนั้นยังมีผู้พัฒนาจากทั่วโลกดำเนินการพัฒนาไลบรารีซึ่งช่วยอำนวยความ สะดวกในด้านต่าง ๆ โดยจะเผยแพร่ในรูปแบบของแพ็คเกจต่าง ๆ ซึ่งสามารถติดตั้งเพิ่มเติมได้อีกด้วย

5.ทำงานได้รวดเร็ว

   ภาษาไพธอน ทำงานเร็วที่สุดเมื่อเทียบกับภาษา script ด้วยกัน เช่น php, jsp, asp จะพูดว่า ไพทอน เขียนน้อยได้งานมาก ทำงานเร็วก็ไม่ผิดนัก

จากกราฟแสดงให้เห็นถึงความนิยมในการใช้ภาษาไพธอนมากขึ้นในปัจจุบัน
