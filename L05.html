Introduction to functions

     คือ กลุ่มของคำสั่งที่มีอยู่ในโปรแกรมเพื่อวัตถุประสงค์ในการปฏิบัติงานเฉพาะ

รู้จักกับฟังก์ชัน

- ส่วนของโค้ดที่ใช้ซ้ำได้ซึ่งทำหน้าที่ดำเนินการ

- ฟังก์ชันมีชื่อและเรียกหรือดำเนินการตามชื่อนั้น

- ฟังก์ชันสามารถยอมรับอาร์กิวเมนต์และส่งคืนข้อมูลได้

Functions Name

กฏการตั้งชื่อฟังก์ชัน

- ไม่สามารถใช้คำสำคัญเป็นชื่อฟังก์ชันได้

- ไม่สามารถมีช่องว่าง

- อักขระตัวแรกต้องเป็นตัวอักษรหรือขีดล่าง

- อักขระอื่นๆทั้งหมดต้องเป็นตัวอักษรตัวเลขหรือขีดล่าง

- อักขระตัวพิมพ์ใหญ่และตัวพิมพ์เล็กแตกต่างกัน

  •      ประโยชน์ของการปรับโปรแกรมด้วยฟังก์ชัน

    •     Simpler Code

          :เข้าใจง่ายขึ้นเมื่อแบ่งออกเป็นฟังก์ชัน

    •     Code Reuses

          :เขียนครั้งเดียวเพื่อดำเนินการนั้นแล้วสามารถเรียกใช้ในเวลาใดๆที่ต้องการ

    •     Better Testing

          :สามารถทดสอบแต่ละฟังก์ชันในโปรแกรมทีละรายการแก้ไขข้อผิดพลาดได้ง่ายขึ้น

    •     Faster Development

          :สามารถรวมไลบรารีของฟังก์ชันในแต่ละโปรแกรมได้

    •     Easier Facliltation of teamwork

          :โปรแกรมเมอร์ที่แตกต่างกันสามารถมอบหมายงานในการเขียนหน้าที่ต่างๆได้ฟังก์ชั่น

Local Variables

     คือ ตัวแปรที่ประกาศภายในฟังก์ชันหลัก หรือการใช้งานได้เฉพาะฟังก์ชันที่ประกาศไว้เท่านั้น

Global Variable

     คือ ตัวแปรที่ประกาศไว้ภายนอกฟังก์ชันหลัก ซึ่งฟังก์ชันอื่น ๆ ก็สามารถเรียกใช้งานตัวแปรแบบนี้ได้

Passing Argument

     คือ ชิ้นส่วนของข้อมูลที่ส่งผ่านไปยังฟังก์ชันเมื่อมีการเรียกใช้ฟังก์ชันพารามิเตอร์ คือ ตัวแปรที่ได้รับอาร์กิวเมนต์ที่ส่งผ่านไปยังฟังก์ชัน
