เนื้อหาบทเรียนเบื้องต้น

รายละเอียดที่ได้เรียนรู้ในแต่ละบท

L00 Introduction

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับการซิงค์ GitHup การติดตั้ง Python และระบบต่างๆ

L01 Data types, variable and expression

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ Computer Programming อธิบายการทำงานของคอมพิวเตอร์ software data อธิบายการทำงานของโปรแกรม CPU และภาษาคอมพิวเตอร์ อธิบายการวาด Flowchart การเขียน Variable Number Boolearn Python operators และ Comment

L02 IF - Elif - Else, Draw.io

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับการติดตั้งและการใช้โปรแกรม Draw.io การใช้ Flowchart Symbols การใช้ if, elif, else รู้จัก Relational operators ที่สำคัญในการเขียนโปรแกรม รู้จักการใช้ String comparison

L03 Repetition While For

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ Repetition Structure การใช้loop While, For การใช้ sentinel, pass, continue, break

L04 String

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ string การใช้ String indexing, String slicing, String formatting, len(), string(), String Methods, String Boolean Method

L05 Function

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ Functions การออกแบบโปรแกรมในการใช้ฟังก์ชัน Passing Arguments to Functions, Local, Global Variables and Global Constants, Value-Returning Functions, Storing Functions in Modules

L06 Files and exceptions

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ File Input and Output, Using Loop to Process Files, Processing Records, Exceptions

L07 Lists and tuples

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ Lists,OperatorList Methods and Useful Built-in Functions, Copying and Processing Lists, Two dimensional Lists, Tuples, mathplot

L08 Dictionary and set

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ Dictionaries การใช้ Sets, Serialising Objects

L09 PyGame-Zero

ในบทเรียนนี้จะได้เรียนรู้เกี่ยวกับ การติดตั้ง pgzero การออกแบบโปรแกรมโดยใช้ pgzero
