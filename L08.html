Dictionaries

    Dictionary : วัตถุที่เก็บชุดข้อมูล

         - แต่ละองค์ประกอบ ประกอบด้วยคีย์และค่า

              - มักเรียกว่าการจับคู่คีย์กับค่า

              - คีย์ต้องเป็นวัตถุที่ไม่เปลี่ยนรูป

         ในการดึงค่าเฉพาะให้ใช้คีย์ที่สัมพันธ์กับค่านั้น

    รูปแบบการสร้าง Dictionaries

    Dictionary = {key1:val1, key2:val2}

Add and Delete dictionary

Dictionaries : เป็นวัตถุที่เปลี่ยนแปลงได้

ในการเพิ่มคู่คีย์- ค่าใหม่ : dictionary[key] = ค่า

     หากมีคีย์อยู่ใน dictionary ค่าที่เกี่ยวข้องกับคีย์จะถูกเปลี่ยน

วิธีลบคู่คีย์- ค่า : del dictionary [key]

     หากคีย์ไม่อยู่ในdictionaryข้อยกเว้นของ KeyError จะเพิ่มขึ้น

    Dictionary number of elements and mixing data type

ฟังก์ชัน len: ใช้เพื่อรับจำนวนองค์ประกอบใน dictionary

keys ต้องเป็นอ็อบเจ็กต์ที่ไม่เปลี่ยนรูปแต่ค่าที่เกี่ยวข้องสามารถเป็นอ็อบเจกต์ประเภทใดก็ได้

     dictionary หนึ่งรายการสามารถรวมคีย์ของประเภทที่ไม่เปลี่ยนรูปแบบต่างๆได้
