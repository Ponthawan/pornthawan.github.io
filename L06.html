File introduction

File input/output

- เพื่อให้โปรแกรมเก็บข้อมูลระหว่างเวลาที่รันคุณต้องการบันทึกข้อมูล

     ข้อมูลจะถูกบันทึกลงในไฟล์โดยทั่วไปแล้วจะอยู่บนดิสก์ของคอมพิวเตอร์

     ข้อมูลที่บันทึกไว้สามารถเรียกค้นและใช้งานได้ในภายหลัง

- "Writing data to":บันทึกข้อมูลลงในไฟล์

- Output file:ไฟล์ที่เขียนข้อมูลถึง

- "Reading data from":กระบวนการดึงข้อมูลจากไฟล์

- Input file:ไฟล์ที่อ่านข้อมูลที่อ่าน

Type of files

ประเภทของไฟล์โดยทั่วไปไฟล์ 2 ประเภท

     1.Text file : มีข้อมูลที่เข้ารหัสเป็นข้อความ 2.Binary file : มีข้อมูลที่ยังไม่ได้แปลงเป็นข้อความ

2 วิธีในการเข้าถึงข้อมูลที่จัดเก็บไว้ในไฟล์

    1.Sequential access:ไฟล์อ่านตามลำดับตั้งแต่ต้นจนจบไม่สามารถข้ามไปข้างหน้าได้

    2.Direct access:สามารถข้ามไปยังข้อมูลใดๆในไฟล์ได้โดยตรง

File

    มี 3 ขั้นตอนเสมอที่ต้องทำเมื่อไฟล์ถูกใช้โดย a โปรแกรม

Open the file     - การเปิดไฟล์จะสร้างการเชื่อมต่อระหว่างไฟล์กับโปรแกรมการเปิดไฟล์เอาต์พุตมักจะสร้างไฟล์บนดิสก์และอนุญาตให้โปรแกรมเขียนข้อมูลลงไปการเปิดไฟล์อินพุตทำให้โปรแกรมสามารถอ่านข้อมูลจากไฟล์ได้

Process the file     - ในขั้นตอนนี้ข้อมูลจะถูกเขียนไปยังไฟล์ (หากเป็นไฟล์เอาต์พุต) หรืออ่านจากไฟล์ (หากเป็นไฟล์อินพุต)

Close the file     - เมื่อโปรแกรมใช้งานไฟล์เสร็จแล้วไฟล์จะต้องปิดลงการปิดไฟล์จะยกเลิกการเชื่อมต่อไฟล์จากโปรแกรมไฟล์
