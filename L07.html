Introduction to lists

     tuplesเป็นลำดับที่สามารถเก็บข้อมูลประเภทต่างๆได้ ความแตกต่างระหว่าง list และ tuples นั้นง่ายมาก:

List   เปลี่ยนแปลงได้ซึ่งหมายความว่าโปรแกรมสามารถเปลี่ยนเนื้อหาได้

Tuple   นั้นไม่สามารถเปลี่ยนแปลงได้ซึ่งหมายความว่าเมื่อสร้างแล้วเนื้อหาจะไม่สามารถเปลี่ยนแปลงได้

List

     เป็นวัตถุที่มีข้อมูลหลายรายการ เปลี่ยนแปลงได้ซึ่งหมายความว่า เนื้อหาสามารถเปลี่ยนแปลงได้ในระหว่างการดำเนินการของโปรแกรมรายการเป็นโครงสร้างข้อมูลแบบไดนามิก หมายความว่า รายการอาจถูกเพิ่มหรือลบออกจากรายการคุณสามารถใช้การทำดัชนีการแบ่งส่วนและวิธีการต่างๆเพื่อทำงานกับรายการในโปรแกรมรายการ

List : วัตถุข้อมูลหลายรายการ

องค์ประกอบ : วัตถุต่างๆในรายการ

รูปแบบ : list = [ item1, item2, etc.]

สามารถใส่ของได้หลายประเภท

print ฟังก์ชันการพิมพ์สามารถใช้แสดงรายการทั้งหมดได้

list() ฟังก์ชั่นสามารถแปลงวัตถุบางประเภทเป็นรายการ

List Methods and Useful Built-in Functions

Repetition operator : ทำสำเนาหลายชุดของรายการและรวมเข้าด้วยกัน

     : สัญลักษณ์ * เป็นโอเปอเรเตอร์การทำซ้ำเมื่อใช้กับลำดับและจำนวนเต็ม

          ลำดับ คือ ตัวถูกดำเนินการทางซ้ายตัวเลขอยู่ทางขวารูปแบบทั่วไป

Genaral format : list * n

รูปแบบ : for x in list:

ข้อยกเว้นIndexError จะเพิ่มขึ้นหากใช้ดัชนีที่ไม่ถูกต้อง

ฟังก์ชันLen : ส่งกลับความยาวของลำดับ เช่น list

     ตัวอย่าง: size = len( my_ list)

ส่งกลับจำนวนองค์ประกอบในรายการ ดังนั้น ดัชนีขององค์ประกอบสุดท้ายคือ len(list)-1

สามารถใช้เพื่อป้องกันข้อยกเว้น IndexError เมื่อวนซ้ำรายการที่มีลูป

สาระเพิ่มเติม

  • append(item)

    : เพื่อเพิ่มรายการลงในรายการ -item ถูกต่อท้ายรายการที่มี

  • index(item)

    : ใช้เพื่อกำหนดตำแหน่งของในรายการส่งกลับดัชนีขององค์ประกอบแรกในรายการที่มีรายการเพิ่มข้อยกเว้น valueError หาก item ไม่อยู่ในรายการ

  • insert(index,item)

    : ใช้แทรกรายการที่ดัชนีตำแหน่งในรายการ

  • sort()

    : ใช้เพื่อจัดเรียงองค์ประกอบของรายการตามลำดับจากน้อยไปมาก

  • remove(item)

    : ลบรายการที่เกิดขึ้นครั้งแรกในรายการ

  • reverse()

    : กลับลำดับขององค์ประกอบในรายการ

  • del statement

    : ลบองค์ประกอบออกจากดัชนีเฉพาะในรายการ

         รูปแบบทั่วไป: del list[i]

  • ฟังก์ชัน min และ max

    : ฟังก์ชันในตัวที่ส่งคืนรายการที่มีค่าต่ำสุดหรือสูงสุดในลำดับถูกส่งผ่านเป็นอาร์กิวเมนต์
